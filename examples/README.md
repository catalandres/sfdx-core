## Examples

- [Display Defaults](./examples/displayDefaults)
- [Alias Editor](./examples/aliasEditor/)

## Setup

Run `yarn install`.

Use `ts-node` to run from the examples directory.
Use `node` to run from the compiled js in the lib directory after running `tsc`.
